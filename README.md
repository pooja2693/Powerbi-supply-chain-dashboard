# 📦 Supply Chain & Inventory Dashboard

An interactive **Power BI dashboard** built to analyze inventory performance, supplier efficiency, warehouse operations, and business KPIs. This project demonstrates end-to-end Business Intelligence using **SQL, Power Query, DAX, and Power BI**.

---

# 📌 Project Overview

The dashboard provides insights into supply chain operations by monitoring inventory levels, supplier performance, product profitability, and shipping efficiency.

It consists of two interactive dashboard pages:

### 📊 Page 1 – Overview Dashboard
- Executive KPI Cards
- Inventory Status
- Supplier Shipping Performance
- Top 10 Profitable Products
- Top Selling Products
- Key Business Insights

### 📈 Page 2 – Detailed Analysis
- Products Needing Restocking
- Supplier Delivery Performance
- Monthly Revenue Trend
- Stock vs Units Sold Analysis
- Supplier Sales Performance

# 📂 Dataset Information

The dataset contains approximately **15,000 records** including:

- Product ID
- Product Name
- Category
- Supplier Name
- Order Date
- Delivery Date
- Stock Quantity
- Reorder Level
- Units Sold
- Purchase Cost
- Selling Price
- Warehouse Location
- Shipping Time (Days)

- 📦 Total Units Sold

# 💼 Business Questions Solved

- Which products require immediate restocking?
- Which suppliers have the highest delivery delays?
- Which products generate the highest profit?
- Which products have the highest sales volume?
- How does monthly revenue change over time?
- Which suppliers perform best based on delivery time?
- How can inventory levels be optimized?

---

# 🔍 Key Insights

- Camera generated the highest revenue and profit.
- Supplier C recorded the fastest average shipping time.
- Products requiring immediate restocking were identified using stock quantity and reorder level.
- Monthly revenue trends highlighted seasonal fluctuations.
- Inventory levels were compared against units sold to support inventory optimization.
- Supplier performance was evaluated using delivery time and sales metrics.
